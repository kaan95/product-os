# Agents

This folder contains specialized AI agents designed to assist with specific product management tasks.

## Structure

Agents are organized by their primary function:

- **discovery-agents**: Agents focused on product discovery activities (assumptions testing, user research, opportunity assessment)
- *(Future folders to be added as needed)*

## How to Use Agents

Each agent is defined in a markdown file that specifies:
- The agent's purpose and capabilities
- Required and optional inputs
- Step-by-step instructions for the agent to follow
- Expected outputs

Agents can be invoked by Claude Code when working on relevant tasks, or you can reference them directly when you need specialized assistance with product management activities.
