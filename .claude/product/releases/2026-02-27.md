# Sprint 32: Release Notes (Frontend)

**Release Date:** 2026-02-27

## Summary

This frontend release introduces AI-powered features for case handling, including feedback collection on AI summaries and AI email summaries in the actions history. It also brings design improvements to the appointment management workflow, adds preview support for email file formats (.eml/.msg), and includes several bug fixes.

## Changes by Epic

### Design Improvements

- **CF-3726**: Redesigned the move/cancel appointment workflow with an improved UI that properly supports rescheduling court appointments and notifies lawyers upon completion.

### AI: Increase Avg. Cases Handled per Lawyer

- **OS-2993**: Added thumbs up/down feedback buttons on AI-generated summaries, allowing lawyers to rate quality and optionally leave notes, helping the team track and improve AI performance.
- **OS-3023**: Replaced the basic email preview (subject + first sentence) in the actions history with AI-generated summaries, so users can quickly understand email content at a glance without opening each one.

### Other

- **CF-3922**: Fixed the task tab in case view to hide the task overview from OS users and remove the completed button for finished tasks.
- **CF-3950 / OS-3084**: Added in-app preview support for .eml and .msg email files, allowing users to inspect email content (sender, recipients, subject, body) directly in the file viewer without downloading.
- **OS-3127**: Disabled PostHog session recording encryption to improve debugging and session replay analysis capabilities.
